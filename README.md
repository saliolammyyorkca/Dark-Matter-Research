# Dark-Matter-Research
Research code for dark force mediator phenomenology and MCMC parameter fitting (York University, Summer 2026).

This repository contains code I developed as part of a theoretical particle physics research project with Dr. Sean Tulin at York University (May–August 2026).

## Contents

### Branching Ratio Calculations
Computes branching ratios for a parameterized dark force mediator across the relevant mass range, including hadronic and leptonic decay channels.

### MCMC Parameter Estimation
Implements Markov Chain Monte Carlo (MCMC) methods to fit a non-perturbative photoproduction model to experimental data and estimate model parameters.
